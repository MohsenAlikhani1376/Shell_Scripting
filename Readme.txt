Commit : Searches for the file (by its name) in the input directory and its subdirectories and returns its count.
------------------------------------------------------------------------------------------------------------------

you should run this script like follow :
./finder-Mohsen <desire directory> <file name

*******************************************************************************************************************

Note 1 : mkdir -p
This command is most helpful in the case when you don't know whether a directory alredy exists or not.
you can create sub-directories of a directory. It will create parent directory first, if it doesn't exist. But if it
already exists, then it will not print an error message and will move further to create sub-directories.
